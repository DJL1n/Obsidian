# 🧠 HermesBrain

这个 vault 是 **Hermes 与你协作的第二个大脑** — 记录想法、项目判断、研究日志、工作风格和知识沉淀。

## 📂 目录结构

```
📥 00_Inbox/          ← 收件箱：所有新笔记的入口
🚀 10_Projects/       ← 活跃项目（有终点）
🗺️ 20_Areas/          ← 领域（持续关注，无终点）
📅 30_Daily/          ← 日记 / 日志（按日期）
📚 40_Knowledge/      ← 知识库（永久笔记）
   ├─ Concepts/       概念笔记
   ├─ Methods/        方法论
   └─ References/     参考资料
✍️ 50_Writing/        ← 写作 / 输出
🧠 60_WorkingStyle/   ← 工作风格 & 元认知
🗄️ 90_Archive/       ← 归档
   ├─ Projects/       已归档项目
   └─ Inbox/          已处理收件箱
```

## 📜 规则

1. **不覆盖旧笔记** — 永远保留历史
2. **新想法先进 Inbox** — 00_Inbox → 分类 → 归档
3. **项目判断进 Projects** — 每个项目一个子目录
4. **工作风格进 WorkingStyle** — 帮助 Hermes 理解你
5. **定期回顾** — 清空 Inbox，归档完成的项目

> 每个目录的 INDEX.md 都有详细说明，用 Obsidian 打开后从 [[00_Inbox/INDEX]] 开始探索。
