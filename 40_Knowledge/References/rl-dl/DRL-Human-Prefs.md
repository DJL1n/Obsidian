# Deep Reinforcement Learning from Human Preferences

> arXiv:1909.08210 | 2019

## 核心贡献

提出通过人类偏好数据直接训练深度强化学习智能体的方法，将 RL 与人类反馈结合，解决 RL 中奖励函数设计的难题。传统 RL 需要精确设计的奖励函数，在复杂任务中往往难以手工构造（reward engineering）。该方法通过人类对智能体行为片段的偏好比较，自动学习到与人类意图一致的奖励函数，进而训练出高性能策略，无需人工设计奖励。

## 方法

该方法采用两个阶段的训练流程：
1. **偏好数据收集**：生成智能体在环境中的多个行为片段（rollouts），让人类标注者对成对的行为片段进行比较，判断哪个更符合目标。使用主动学习策略选择最有帮助的样本对进行标注，减少标注成本
2. **奖励模型训练**：将偏好数据形式化为 Bradley-Terry 模型，训练一个基于深度神经网络的奖励函数 r(s,a)，使其预测的奖励与人类偏好一致
3. **策略优化**：使用学习到的奖励函数训练策略，可以采用离线方法（从已有数据中学习）或在线方法（继续与环境交互）

论文在 MuJoCo 和 2D 网格世界任务上验证，证明从人类偏好学习的奖励函数能产生比手工奖励更好的策略。

## 影响

该工作将人类反馈引入 RL 领域，为 RLHF 的前身提供了理论基础。它展示了偏好学习（preference learning）在 RL 中的有效性，启发了后续的对齐研究（如 InstructGPT、RLAIF）。方法中的主动学习和 Bradley-Terry 偏好模型被广泛应用于后续的偏好优化算法。这项工作桥接了经典 RL 和 LLM 对齐，证明人类偏好作为奖励信号可以替代或增强传统奖励函数设计。
