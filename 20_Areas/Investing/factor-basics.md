# 因子投资基础（Factor Investing）

## IC（Information Coefficient）

```
IC_t = corr(因子值_t, 下期收益率_{t+1})
```

| IC | 含义 |
|---|---|
| > 0.05 | 强因子 |
| 0.02-0.05 | 中等因子 |
| 0.01-0.02 | 弱因子 |
| < 0.01 | 几乎无效 |

IC 是衡量因子预测能力的核心指标。

## IR（Information Ratio）

```
IR = E(IC) / σ(IC)
```

衡量因子稳定性：
| IR | 含义 |
|---|---|
| > 1.0 | 极好 |
| 0.5-1.0 | 好 |
| 0.2-0.5 | 一般 |
| < 0.2 | 不稳定 |

## 分组回测（Quantile Portfolio）

把股票按因子值分 N 组，看各组收益差异：

```
Group 1（因子最低） → Group 2 → ... → Group 5（因子最高）

因子有效 → Group 5 收益 > Group 1 收益（单调递增）
```

分组回测比 IC 更直观。

## 因子衰减

因子有效性会随时间衰减：
- 市场学习到因子 → 套利 → 因子失效
- 新因子需要持续研发
- 经典因子（价值/动量）衰减慢但收益也低
- 新因子可能衰减快

## 因子中性化

去掉因子的市场/行业暴露，只看纯 alpha：

```
中性化因子 = 因子 - β_行业 × 行业因子 - β_市值 × 市值因子
```

## 多因子合成

多个弱因子组合成强信号：

```
综合得分 = w₁ × 标准化(因子₁) + w₂ × 标准化(因子₂) + ...
```

权重可以用 IC-IR 加权（IR 高的因子给更高权重）。
