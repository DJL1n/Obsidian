# 量化交易本质与因子

## 量化是什么

量化交易 = 用数学模型做投资决策。交叉学科：

| 学科 | 在量化中的应用 |
|---|---|
| 统计学 | 因子建模、回归、时间序列 |
| 概率论 | 随机过程、鞅论、蒙特卡洛 |
| 最优化 | 组合优化、均值-方差 |
| 信息论 | 因子独立性、信号熵 |
| 机器学习 | 非线性因子发现 |

## 因子（Factor）

因子 = 用来预测股票收益的变量

### 经典因子

- **价值**：低 P/E、低 P/B → 长期跑赢（Fama-French）
- **动量**：过去 N 个月涨的股票继续涨（Jegadeesh & Titman, 1993）
- **质量**：高 ROE、低负债、盈利稳定
- **低波动**：低波动率股票长期风险调整后收益更高（volatility anomaly）
- **规模**：小盘股长期跑赢（A 股近年失效）

### 因子为什么重要

超额收益分解：
```
超额收益 = 因子暴露 × 因子收益 + 交易成本 + 噪音
```

因子是 alpha 的唯一源头——没有因子，就没有超额收益。

### 因子衰败（Alpha Decay）

市场是竞争的。因子被广泛使用后，超额收益被"套利走"：
- 经典因子 1990 年代有效，现在大幅衰减
- 私募量化核心竞争力 = 因子研发能力
- 需要不断挖掘新因子（另类数据、机器学习发现）

## 因子之外的关键

因子是原材料，但不是全部：

1. **因子合成** — 多个弱因子组合成强信号（集成学习思想）
2. **换手率控制** — 信号变化太快 → 交易成本吃掉收益
3. **风控** — 单因子失效时（如 2021 小盘崩盘）的应对措施
4. **执行** — 滑点、冲击成本、算法交易

## 核心观点

> 因子是量化交易的"食材"，合成+风控+执行是"烹饪"。两者缺一不可。


## 金融建模的本质

量化 = 对金融市场这个"经济体"建数学模型。但金融市场不同于物理系统：

| 特性 | 物理系统 | 金融市场 |
|---|---|---|
| 规律稳定性 | 稳定（重力不变） | **会变的**（参与者学习后规律失效） |
| 可预测性 | 高 | 低（噪声极大） |
| 反馈 | 无 | **有**（预测对了 → 别人复制 → 预测失效） |

### 建模的五个层次

1. **统计建模** — 回归、因子、时间序列。假设历史模式重复。**问题：市场会变**
2. **均衡模型** — CAPM、APT、Fama-French。假设理性人+市场均衡。**问题：人不是理性的**
3. **行为建模** — 认知偏差（过度自信、损失厌恶、羊群效应）。**问题：难以量化**
4. **博弈建模** — 市场微观结构、订单流。**问题：太复杂**
5. **机器学习建模** — 不假设机制，直接从高维数据找模式。**问题：过拟合风险极大**

### 核心矛盾

> 如果你能完美建模市场，你就是市场。  
> 如果你能赚钱，别人会复制你，你的优势消失。

量化机构的壁垒不是模型复杂度，而是**持续发现新偏差的能力**——这就是"因子"。

### 关键结论

> 量化不是建模"均衡"，而是建模**经济体的非均衡偏差**。  
> 偏差一旦被建模成功并广泛使用，就不再是偏差了。


## Meta-Level 博弈

### 因子衰败的本质

```
发现因子 → 赚钱 → 别人发现 → 一起用 → 因子失效
```

量化机构的核心壁垒 = 持续发现新偏差的能力。但偏差被广泛使用后消失。

### Meta-Strategy（量化的量化）

不只建模市场，而是**建模其他量化策略的行为**：
- 预测因子何时拥挤、何时失效
- 因子拥挤度指标：当太多基金持有同一因子时收益下降
- 反转操作：在因子被过度使用后反向做空
- 量化踩踏：2015 年 A 股量化基金集体清仓引发连锁下跌
- 高频反制：检测到订单模式后"截胡"

### 博弈层级与利润递减

```
层级1: 散户 ← 被机构割
层级2: 机构 ← 被量化割
层级3: 量化 ← 被 meta-quant 割
层级4: meta-quant ← 被 meta-meta-quant 割
```

每一层：信号越微弱、噪声越大、计算越昂贵。**最高层 alpha 接近于零。**

### 对个人投资者的启示

- 个人直接做 meta-quant 不现实（资源不对称）
- 场外基金持有人已处于"被打包"位置
- 对手不是量化公司，而是：市场波动 + 基金经理选择 + 费率
- **最优策略：选对博弈的参与者（基金和经理），而非加入博弈**


## 主动基金 vs 量化基金

### 主动基金管理结构

```
基金经理（PM）— 决策买什么、仓位
  ↓
研究团队 — 行业研究、个股调研
  ↓
量化团队（如有）— 因子筛选、风险模型、组合优化
  ↓
交易员 — 算法执行（控制冲击成本）
```

大部分主动基金也用量化工具（Barra 风险模型、因子筛查、算法交易），但 PM 是最终决策者。

### 散户赚钱来源

| 来源 | 性质 |
|---|---|
| 政策 | 系统性信号（宏观变化） |
| 第一手消息 | 信息优势（少数人先知道） |
| 运气 | 纯噪声（随机波动幸存者偏差） |
| 情绪 | **行为偏差**（可被建模的噪声） |

**关键区别**：运气不可预测，但情绪是可预测的偏差。量化策略利用散户情绪偏差赚钱。

## 噪声的层级结构

```
层级1: 单个散户随机决策 → 独立同分布噪声
层级2: 散户群体情绪共振 → 相关噪声（羊群效应）
层级3: 机构对散户反应 → 反馈噪声
层级4: 多机构博弈 → 博弈噪声
```

噪声在不同层级**不是独立的**——正反馈回路会放大噪声（2015 A股崩盘、2020美股熔断）。

## 噪声建模

### 标准做法：y = f(x) + ε

最基础建模：在函数后加噪声项 ε ~ N(0, σ²)

### 实际量化中的复杂噪声建模

| 方法 | 建模内容 | 复杂度 |
|---|---|---|
| 线性 + 噪声 | 独立同分布 | 最简单 |
| GARCH | 噪声**方差会变化**（波动率聚类） | 中等 |
| 状态空间模型 | 噪声和信号都随时间变化 | 较复杂 |
| 跳跃扩散 | 噪声不连续，有**跳变**（暴跌/暴涨） | 复杂 |
| 重尾分布 | 极端事件比正态分布频繁得多 | 复杂 |

### 关键问题

> 金融市场噪声不服从正态分布。  
> 正态分布假设极端事件罕见，但"黑天鹅"频繁出现。  
> **量化真正的难点不是建模函数 f，而是建模噪声 ε 的分布**——不独立、不平稳、尾部重、会反馈、会放大。


## 因子在公式里的位置

### 核心公式

```
股票超额收益 = Σ(βⱼ × Fⱼ) + ε
```

| 符号 | 含义 |
|---|---|
| Fⱼ | 因子 j 的**收益**（横截面共享） |
| βⱼ | 股票对因子 j 的**暴露度/权重** |
| ε | 残差（模型解释不了的部分） |

### 具体例子

**CAPM（单因子）**：
```
Rᵢ - Rf = βᵢ × (Rm - Rf) + εᵢ
```
唯一因子 = 市场超额收益

**Fama-French 三因子**：
```
Rᵢ - Rf = β_MKT × MKT + β_SMB × SMB + β_HML × HML + εᵢ
```
MKT = 市场，SMB = 小盘-大盘，HML = 价值-成长

### 因子 vs 特征

| | 因子（Factor） | 特征（Feature） |
|---|---|---|
| 粒度 | **横截面**（所有股票共享） | 个股级别 |
| 含义 | 可解释的市场驱动力 | 可以是任何数值 |
| 建模 | `收益 = β × 因子 + ε` | `收益 = f(特征₁, 特征₂, ...) + ε` |

### 比喻

因子 = 风向，β = 帆船受风角度。同一阵风吹过所有船，但帆角度不同，速度不同。
