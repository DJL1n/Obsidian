# 财报与估值基础

## 三大核心指标

### PE（市盈率）
```
PE = 股价 / 每股收益（EPS）
```
含义：多少年回本。
- <10：便宜（或有问题）
- 10-20：合理
- 20-30：稍贵
- >30：贵或高成长预期

### PB（市净率）
```
PB = 股价 / 每股净资产
```
- <1：破净
- 1-2：合理
- >3：贵
适合重资产行业（银行、地产）

### ROE（净资产收益率）
```
ROE = 净利润 / 股东权益
```
**最核心指标**：股东每投入 1 块赚多少。
- >20%：极好（巴菲特标准）
- 15-20%：好
- 10-15%：一般
- <10%：差

## 三指标关系
```
PE × ROE = PB
```

## 指标组合判断

| PE | ROE | 含义 |
|---|---|---|
| 低 | 高 | **买入信号**（便宜又好） |
| 高 | 高 | 好公司但贵，等回调 |
| 低 | 低 | 价值陷阱 |
| 高 | 低 | 不买 |

## 看财报关注什么

### 利润表
- 营业收入（增长吗？）
- 净利润（利润率和收入匹配吗？）
- 扣非净利润（排除一次性收益）

### 资产负债表
- 资产负债率（<60% 较安全）
- 应收账款（占收入比例，越高越危险）
- 存货（周转率，越高越好）

### 现金流量表
- 经营现金流（> 净利润为好）
- 投资现金流（扩张期负正常）
- 筹资现金流（分红/回购为正）

**关键**：经营现金流 ≥ 净利润。如果净利润高但现金流差，可能是"纸面富贵"。
