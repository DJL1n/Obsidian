# 市场情绪周期

> 短线交易的核心框架：冰点 → 启动 → 高潮 → 退潮 → 冰点

## 四个阶段

### 1. 冰点（Fear）
- 特征：连板股大量跌停，成交量萎缩，极度悲观
- 操作：观察，不操作
- 量化信号：涨停 < 20，跌停 > 10，连板高度 ≤ 2

### 2. 启动（Hope）
- 特征：新题材出现，首板/二板增多，情绪转谨慎
- 操作：试仓，找新题材龙头
- 量化信号：涨停 > 40，连板高度 ≥ 3

### 3. 高潮（Euphoria）
- 特征：板块爆发，涨停潮，散户涌入
- 操作：持有龙头，卖出跟风，准备撤退
- 量化信号：涨停 > 80，封板率 > 90%

### 4. 退潮（Disappointment）
- 特征：龙头断板，跟风大跌，炸板率升高
- 操作：清仓，等冰点
- 量化信号：涨停 < 30，炸板率 > 50%

## 为什么龙头抗跌？

```
高潮期：龙头一字板（没人卖），跟风涨停
退潮期：跟风先跑（逻辑弱），龙头后跌（锁仓多）
结果：龙头跌 20-30%，跟风跌 50-70%
```

## 情绪周期 vs 我们的动量因子

| 维度 | 龙头战法 | 动量因子 |
|---|---|---|
| 频率 | 分钟级（打板） | 日频（月度调仓） |
| 仓位 | 全仓 1 只 | 分散 10 只 |
| 成本 | 极高（冲击成本 1-3%） | 低（滑点 0.1%） |
| 风险 | 腰斩可能 | 控制在 15% 内 |
| 可持续性 | 需全职盯盘 | 可自动化 |

动量因子本质上是捕捉"启动→高潮"阶段的收益，但用更温和的方式。
