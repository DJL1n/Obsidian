
## Alpha 和 Beta 的正式区分（CAPM）

```
预期收益 = Rf + β × (市场超额收益)
实际收益 = Beta收益 + Alpha
Alpha = 实际收益 - (Rf + β × (E[Rm] - Rf))
```

Alpha = 扣除风险后的多余收益。Beta 收益是"应得的"，Alpha 是"额外的"。

| Alpha | 含义 |
|---|---|
| > 0 | 跑赢市场（经理有本事） |
| = 0 | 刚好匹配风险 |
| < 0 | 跑输市场（经理不行或费率高） |

## 有效市场假说（EMH）

市场价格已反映所有可用信息 → 不可能持续获得超额收益。

| 强度 | 含义 | 现实 |
|---|---|---|
| 弱式 | 历史价格已反映 | 基本成立 |
| 半强式 | 公开信息已反映 | 部分成立 |
| 强式 | 所有信息已反映 | 不成立 |

**市场介于"完全有效"和"完全无效"之间**——有套利空间，但很快被填平。

## 超额收益的来源

超额收益 ≠ 信息差。信息差只是其中之一：

| 来源 | 机制 | 例子 |
|---|---|---|
| 信息差 | 你知道别人不知道的事 | 内幕消息（违法） |
| 风险补偿 | 承担别人不愿承担的风险 | 买小盘股 |
| 行为偏差 | 利用别人非理性 | 散户恐慌时买入 |
| 计算优势 | 模型比别人准 | 量化因子 |
| 时间优势 | 反应比别人快 | 高频交易 |

**大部分超额收益来自风险补偿 + 行为偏差。**

信息差在越来越小的市场里价值越来越低（互联网+监管披露）。真正赚钱的不是"我知道你不知道的"，而是"我知道你知道的，但理解更深"。

> 世界上没有免费的超额收益，只有你愿意承担的代价。
