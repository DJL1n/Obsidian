# 回测陷阱（Backtest Pitfalls）

> 量化新手最容易死在这里——不是模型不够复杂，而是回测在骗人。

## 1. 未来函数（Lookahead Bias）

最致命的错误——用了未来才能知道的数据。

### 例子
```
错误：1月1日用当月数据算收益率，当天就买卖
正确：1月2日用上月数据，基于已知信息买卖
```

### 检测
- 信号日 < 成交日
- 因子只用 signal_date 之前数据

## 2. 幸存者偏差（Survivorship Bias）

只研究现在还活着的股票，忽略已退市/ST 股票。

### 后果
回测收益虚高 10-20%

### 例子
2000 年 A 股退市了 200+ 只股票 → 如果只回测现存股票 → 漏掉暴跌退市 → 收益虚高

### 检测
标的池必须用当时的成分，不是现在的

## 3. 过拟合（Overfitting）

参数调得太完美，只在历史数据上有效。

### 例子
试 1000 种参数 → 找到"完美"组合（lookback=47, top_n=13, weight=0.57） → 实盘失效

### 检测
- 样本外测试（用未参与优化的数据验证）
- 参数越简单越好

## 4. 多重检验（Multiple Testing）

试得越多，越容易"碰巧"找到有效的。

### 统计学
试 100 个因子，假阳性率 5% → 平均 5 个是碰巧有效的

### 检测
- Bonferroni 校正
- 交叉验证

## 5. 数据 Snooping

反复看回测结果，调整策略，直到看起来"好"。

### 后果
大脑在过拟合，不是策略本身有效

### 正确做法
假设 → 锁定策略 → 一次回测 → 样本外验证

## 审计清单

回测前必须确认：
- [ ] 没有未来函数（信号日 < 成交日）
- [ ] 没有幸存者偏差（用历史成分）
- [ ] 参数数量合理（不过多）
- [ ] 有样本外验证
- [ ] 交易成本已计入（手续费 + 滑点 + 印花税）
- [ ] 停牌/涨跌停已处理
- [ ] 复权正确（前复权 vs 后复权）
