# 仓位管理与止损止盈

## 仓位管理

### 核心原则

1. **永远不满仓** — 保持 20-30% 现金，大跌时能补仓
2. **单只不超 20-30%** — 暴雷时总损失可控
3. **分批买入** — 试探→加仓→再加，避免买在高点
4. **用风险预算控制** — 最大亏损 = 总资金 × 5%

### 风险预算公式

```
单只最大投入 = 总资金 × 最大可接受亏损 / 单只止损比例
```

例：10 万资金，最大接受亏 5%，单只止损 15% → 单只最多 3.3 万

## 止损

| 类型 | 怎么设 | 适合 |
|---|---|---|
| 固定比例 | 亏 8-15% 就卖 | 新手 |
| 技术位 | 跌破均线就卖 | 有经验 |
| 基本面 | 逻辑坏了就卖 | 价值投资 |

**止损核心**：亏 10% → 涨 11% 回本，亏 50% → 涨 100% 回本。越早止损越容易回本。

## 止盈

| 方式 | 说明 |
|---|---|
| 固定目标 | 赚 20% 就卖 |
| 动态止盈 | 涨了止损位跟着上移（最推荐） |
| 分批止盈 | 涨到目标卖一半，另一半继续拿 |

### 动态止盈例子

买入 100 元，止损 -10%，目标 +20%：
- 涨到 110 → 止损上移到 100（保本）
- 涨到 120 → 卖一半
- 涨到 130 → 止损上移到 110
- 跌到 110 → 全部卖出（赚 10%）

**最坏保本，最好赚更多。**

## 核心结论

> 仓位管理控制亏多少，止损不让亏变大，止盈不让赚变亏。
> 这三条比选股更重要。
