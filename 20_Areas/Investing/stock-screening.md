# 选股入门：PE/PB/ROE 筛选

## 三步筛选法

### 1. ROE > 15%（连续 3 年）— 找好公司
一年 ROE 高可能是偶然，3 年才说明竞争力。

### 2. PE < 行业平均 — 找便宜价格
| 行业 | 合理 PE |
|---|---|
| 银行 | 5-8 |
| 地产 | 5-10 |
| 消费 | 15-25 |
| 科技 | 20-40 |
| 医药 | 25-40 |

不能跨行业比 PE。

### 3. PB < 3 — 找安全边际
PB 低 = 股价接近净资产 = 下跌空间有限。
银行/地产看 PB < 1.5。

## 组合筛选器

```
ROE > 15% + PE < 20 + PB < 3 + 市值 > 100 亿
```

通常只有几十只符合。

## 筛选后逐一分析

1. 行业：你能理解吗？
2. 收入增长：> 10% 为佳
3. 负债率：< 60% 安全
4. 经营现金流：是否覆盖净利润？
5. 大股东：有没有减持？

> 如果不能一句话解释公司怎么赚钱 → 不买

## 例子

消费股筛选：
- 伊利：ROE 20%，PE 18，PB 3 → ✓
- 茅台：ROE 30%，PE 30，PB 8 → PE/PB 超（贵但好）
- 小酒厂：ROE 8% → ✗

## 找不到怎么办

- 买指数 ETF（沪深 300/中证 500）
- 等好公司回调
- 买行业 ETF
- **继续定投 QDII + 小仓位 A 股练习**
